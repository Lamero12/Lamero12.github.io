How to Configure Proxy in Your Browser

Google Chrome

Method 1: Using Settings

  1. Open Chrome Settings
  2. Search for "proxy"
  3. Click "Open your computer's proxy settings"
  4. Enter proxy details

Method 2: Using Extension

I recommend Proxy SwitchyOmega extension:

1. Install from Chrome Web Store
2. Click extension icon
3. Add new profile
4. Enter proxy server details:
   - Protocol: HTTP/HTTPS/SOCKS5
   - Server: IP address
   - Port: Port number
5. Save and activate

Mozilla Firefox

Steps:

  1. Open Settings (about:preferences)
  2. Scroll to Network Settings
  3. Click Settings button
  4. Select Manual proxy configuration
  5. Enter proxy details:
    • HTTP Proxy: 192.168.1.1
    • Port: 8080
  6. Check "Use this proxy for all protocols"
  7. Click OK

Example Configuration:

HTTP Proxy: 185.162.228.73
Port: 80
HTTPS Proxy: 185.162.228.73
Port: 443

⚠️ Warning: Always verify the proxy source before using it!

Testing Your Proxy

Visit these sites to verify:

  • https://whatismyipaddress.com
  • https://whoer.net
  • https://ipleak.net

Your IP should show the proxy's location, not your real one.
